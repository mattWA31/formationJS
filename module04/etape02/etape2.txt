Base de l'application

1. Avant de commencer à écrire notre code front-end, nous devons implémenter une base serveur-client à partir de laquelle travailler. Cela signifie qu'une vue HTML de base est servie à partir d'un serveur Express. Pour des raisons de performance et de fiabilité, nous allons injecter les dépendances frontales directement depuis le dossier node_modules. Nous devrons configurer notre serveur Express d'une manière spéciale pour que cela fonctionne. Ouvrez server.js et ajoutez ce qui suit :

const express = require('express');

const app = express();
const port = 3000;

// Définir le dossier public comme racine
app.use(express.static('public'));

// Autorise les clients a accéder aux fichier statiques
app.use('/scripts', express.static(`${__dirname}/node_modules/`));

// Ecouter sur le port 3000
app.listen(port, function() {
  console.log('Ecoute sur %d', port);
});

2. Ensuite, ouvrez public/index.html et entrez :

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <link rel="stylesheet" href="scripts/semantic-ui-css/semantic.min.css">
  <title>Taux Application</title>
</head>
<body>
  <div class="ui container">
    <!-- Menu de navigation -->
    <div class="ui four item inverted blue menu">
      <div class="header item">
        <i class="money bill alternate outline icon"></i>
        Les Taux
      </div>
      <a class="item" href="/">
        Cours des devises
      </a>
      <a class="item" href="/conversion">
        Conversion
      </a>
      <a class="item" href="/historique">
        Historique des taux
      </a>
    </div>

    <!-- Application Root -->
    <div id="app"></div>
  </div>

  <!-- JS Library Dependencies -->
  <script src="scripts/jquery/dist/jquery.min.js"></script>
  <script src="scripts/semantic-ui-css/semantic.min.js"></script>
  <script src="scripts/handlebars/dist/handlebars.min.js"></script>
  <script src="scripts/vanilla-router/dist/vanilla-router.min.js"></script>
  <script src="js/app.js"></script>
</body>
</html>

3. npm start

4. Ouvrez localhost:3000 dans votre navigateur. Vous devriez avoir une page blanche avec seulement la barre de navigation affichée.
